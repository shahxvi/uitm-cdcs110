# Analyse the Problem (IPO)

## Input
category, itemcode, size, quantity

## Process
```
    Check category
        D = Decoration
        L = Landscape

    Check itemcode
        D001 = Birah
        D002 = Bongsai
        L001 = Pinang Merah
        L002 = Pagoda
    
    Check size
        Small
        Medium
        Large
    
    Check price

    Check quantity

    Calculate totalPrice = price * quantity;
```

## Output
price, quantity, totalPrice
